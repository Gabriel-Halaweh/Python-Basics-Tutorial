# Python Tutorials: Basics & Advanced

There are two Google Colab tutorials. Each notebook pairs a plain-language explanation with runnable code and its actual output, so a learner sees cause and effect immediately instead of just reading about a concept.

# What's covered
`Python_Basics.ipynb`
- Comments (single-line and multiline) and indentation/code blocks
- Variables and core data types: integers, floats, strings, booleans, lists, tuples, dictionaries
- Control flow: conditional statements (`if` / `elif` / `else`)
- Loops: `for` loops, `while` loops, and using `range()`
- Functions: creation, parameters, return values, docstrings, naming conventions, function composition, and recursion
- Error handling with `try` / `except` / `else`
- Debugging with `print` statements, including a hands-on example demonstrating floating-point precision quirks
  
`Python_Advanced.ipynb`
- Advanced data types: lists, tuples, sets, dictionaries, and comprehensions (list, set, and dictionary)
- Object-oriented programming: classes and objects, attributes and methods, inheritance, polymorphism, and encapsulation
- Importing and using Python's standard libraries (e.g., `math`, `random`)
- Regular expressions: pattern matching, metacharacters, quantifiers, grouping/capturing, lookahead/lookbehind, and substitution

# Why I built these
Our project needed teammates to ramp up on Python quickly, so I wrote these tutorials to take someone from zero experience to comfortable with core and intermediate concepts. Every section includes a working example and its real output, so learners can test their own understanding against the result before moving on.

# How to use them
Both notebooks are built for Google Colab. Click `Open in Colab` at the top of each file to run the code live in your browser, no setup required.
